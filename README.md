# Collaborative Cursors
### Figma-like collaborative cursor using socketio

![banner](./banner.png)



## Features

- Realtime Cursors 🌐 
- 4000 x 4000 px draggable canvas area 🟩
- Rooms 🏡
- Drawing ✏️
- Stroke width size ⭕
- Colors 🎨
- GIFS ✨
